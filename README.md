## 시작 가이드
### Requirements
For building and running the application you need:
- Node.js 18.16.0
- Npm 9.5.1
- NextJS 13.4.9
- ReactJS 18.2.15
### Installation
```shell
$ git clone
$ cd mInstagram

$ npm install
$ npm run dev
```
